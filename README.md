# Página de Exclusão de Conta - Meu Bebê Prematuro

Esta é uma página independente de exclusão de conta para o aplicativo Meu Bebê Prematuro, hospedada no Cloudflare Pages.

## URLs de Acesso

- **Principal**: https://delete-account-meubebeprematuro.pages.dev
- **Alternativa**: https://meubebeprematuro.pages.dev/delete-account

## Conteúdo da Página

A página apresenta:

-  Informações sobre o que será deletado
-  Guia passo a passo para deletar a conta
-  Detalhes de privacidade e segurança
-  Opção de baixar dados antes de deletar
-  Informações de contato com suporte

## Deploy

Esta página é automaticamente deploiada no Cloudflare Pages sempre que há um push para a branch main.

### Como fazer alterações:

1. Faça alterações no arquivo delete-account.html
2. Commit e push:
   git add . && git commit -m "sua mensagem" && git push origin main
3. O Cloudflare Pages automaticamente redeploya a página

## Estrutura
